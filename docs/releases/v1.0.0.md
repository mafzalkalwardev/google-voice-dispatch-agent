# v1.0.0 — Initial Stable Release

Google Voice AI outbound calling console with FastAPI operator UI.

## Highlights

- Selenium Google Voice control, STT/LLM/TTS loop
- Call logs, CRM enrichment, Windows packaging path
- Expanded compliance section

## Install

```powershell
git clone https://github.com/mafzalkalwardev/google-voice-dispatch-agent.git
cd google-voice-dispatch-agent
pip install -r requirements.txt
python -m src.main --preflight
```

Consent and lawful use required for live calling.
