What Each Check Validates

  • ENV File — .env exists and GROQ_API_KEY is set (not placeholder).
  • Groq API — Live HTTP call to Groq; confirms key is valid and returns model list.
  • Contacts File — Configured contacts path exists and can be parsed by load_contacts().

  • Callback Number — CALLBACK_NUMBER is configured in .env or dialer_config.json.
